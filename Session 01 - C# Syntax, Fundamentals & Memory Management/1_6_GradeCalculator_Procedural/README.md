⚠️  PROBLEMS WITH PROCEDURAL APPROACH
══════════════════════════════════════════════════
1. Data is scattered across multiple places
2. Easy to make mistakes (index mismatches)
3. Hard to add new properties (need new array each time)
4. No encapsulation - anyone can modify data
5. Functions need many parameters
6. Difficult to maintain and scale
7. No clear relationship between related data
